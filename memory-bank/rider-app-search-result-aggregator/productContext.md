# Product Context

## Why This Project Exists
- To aggregate search results from multiple sources for the rider app.

## Problems It Solves
- Provides a unified view of available rides from different providers.

## How It Should Work
- The rider-app/search-result-aggregator service should collect search results from various sources and present them to the rider app.
