# Project Brief

## Project Goals
- Define the core requirements and goals of the dynamic-offer-driver-app project.
- Serve as the source of truth for project scope.

## Core Requirements
- To provide API services for dynamic offers and driver management.
- To handle driver onboarding, including identity and document verification (DL, RC, PAN, GSTIN, Aadhaar).
- To manage payment and payout processes for drivers.
- To integrate with various external services via webhooks (e.g., Idfy, HyperVerge, Juspay, Safety Portal).
- To expose API documentation via Swagger/OpenAPI.
