# Project Brief

## Project Goals
- Define the core requirements and goals of the dynamic-offer-driver-drainer project.
- Serve as the source of truth for project scope.

## Core Requirements
- To provide asynchronous data synchronization between Redis (KV store and Stream) and PostgreSQL for dynamic offer and driver-related data.
- To ensure data consistency and fault tolerance during data persistence for this specific domain.
- To optionally push dynamic offer and driver data to Kafka for analytical purposes.
