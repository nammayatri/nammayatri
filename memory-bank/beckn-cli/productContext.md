# Product Context

## Why This Project Exists
- To provide a command-line interface for developers and operators to interact with the Beckn network.

## Problems It Solves
- Simplifies the process of sending requests to the Beckn network.
- Provides tools for generating key pairs and performing load testing.
- Used for initiating load tests using the K6 framework.

## How It Should Work
- The beckn-cli tool should provide commands for generating key pairs, sending requests, and performing load tests.
- It uses the K6 framework for load testing.
- It can generate public/private key pairs for the Registry using the Ed25519 algorithm.

## User Experience Goals
- Easy to use and configure.
- Provides clear and informative output.
- Supports a wide range of Beckn API operations.
