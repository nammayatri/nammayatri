# Product Context

## Why This Project Exists
- To automate the generation of code for the NammaYatri platform.

## Problems It Solves
- Reduces the amount of boilerplate code that needs to be written manually.
- Ensures consistency across different parts of the codebase.
- Simplifies the process of creating new APIs and data models.

## How It Should Work
- The alchemist tool should take a YAML specification as input and generate Haskell code, SQL queries, and other artifacts.

## User Experience Goals
- Easy to use and configure.
- Generates high-quality, maintainable code.
- Supports a wide range of code generation scenarios.
